# Code of Conduct

This project follows a simple standard: be direct, respectful, and focused on improving the software.

Participants are expected to:

- Discuss technical disagreements with evidence and clear reasoning.
- Keep issues and pull requests relevant to the project.
- Avoid harassment, personal attacks, or discriminatory language.
- Respect maintainers' decisions about scope, compatibility, and project direction.

Unacceptable behavior may result in comments, issues, or pull requests being closed.
